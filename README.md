**WSJT-X 3.0.1 modified for ATNi**

This modification of WSJT-X is designed for use by **Islands on the Air (IOTA) DXpeditions**.

In Fox (or Super Fox) mode, calls that are not on a list of IOTA club members are ignored, as an option.

The purpose is to allow club members to obtain their **All-Time-New-Island (ATNi)** QSOs preferentially over all other callers.

The file _IOTA-HRAL.txt_ must be included in the WSJT-X log directory. That file is a CRLF-delimited list of IOTA club members.
